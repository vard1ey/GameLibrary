# GameLibrary

Данное мобильное приложение разработано для систематизации коллекции видеоигр. Приложение предназначено для геймеров, которые хотят отслеживать свой прогресс в прохождении игр.

## Использование
Для работы с приложением необходимо следующее программное обеспечение:
* Устройство под управлением Android 5.0 (API 21) или выше
* Эмулятор Android в среде разработки Android Studio.

Перед началом работы с Игровой библиотекой необходимо выполнить следующие действия:
* Запустите приложение "Game Library" на устройстве.
* Если вы новый пользователь, нажмите "Нет аккаунта? Зарегистрироваться".
* Заполните форму регистрации (Имя, Email, Пароль) и нажмите "Зарегистрироваться".
* После успешной регистрации войдите в систему, используя свои учетные данные.
* На главном экране библиотеки нажмите "+ Добавить игру", чтобы начать формировать свою коллекцию.

### Добавление игры
1. Введите название игры в поле "Название игры"
2. Выберите жанр из выпадающего списка "Жанр"
3. Выберите платформу из выпадающего списка "Платформа"
4. Для платформы "Mobile" появится кнопка "Выбрать приложение" - нажмите её для выбора установленной игры
5. Добавьте заметки (необязательно)
6. Нажмите "Добавить игру"

### Управление играми в библиотеке
* **Изменение статуса:** Нажмите на карточку игры для циклического изменения статуса: "Не начата" → "В процессе" → "Завершена"
* **Запуск мобильной игры:** Для игр на платформе "Mobile" нажмите на иконку геймпада в карточке игры
* **Удаление игры:** Нажмите кнопку "✕" в правом верхнем углу карточки игры
* **Выход из аккаунта:** Нажмите кнопку "Выйти" внизу главного экрана

## FAQ
### Где найти кнопку для добавления игры?
* Она находится в верхней части главного экрана "Библиотека игр" и имеет надпись "+ Добавить игру".

### Как изменить статус прохождения игры?
* Просто нажмите на карточку с игрой в библиотеке. Статус будет циклически меняться: "Не начата" → "В процессе" → "Завершена".

### Как удалить игру из библиотеки?
* Нужно нажать кнопку "✕" в правом верхнем углу карточки игры.

### Как запустить мобильную игру из приложения?
* Для игр на платформе "Mobile" нажмите на иконку геймпада в карточке игры. Убедитесь, что приложение для запуска было выбрано при добавлении игры.
